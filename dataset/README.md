# О датасете

## Информация о данных: 
* Датчик WISDM (WIireless Sensor Data Mining), работающий на основе смартфона и собирающий данные от 36 разных пользователей в шести различных видах деятельности.
* Количество примеров: 1 098 207
* Количество атрибутов: 6
* Отсутствующие значения атрибутов: Нет

## Обработка данных:
1. Замените наносекунды на секунды в столбце «Временная метка» и удалите столбец «Пользователь», поскольку каждый пользователь будет выполнять одно и то же действие.
2. Используйте метод скользящего окна для преобразования данных в последовательности, а затем разделите каждую метку на обучающую и тестовую выборки, обеспечив соотношение 8:2 для каждой метки в обеих выборках.
3. Перемешайте порядок меток в обучающей и тестовой выборках, чтобы предотвратить последовательное появление двух последовательностей с одинаковой меткой.

## Активность:
* 0 = Спуск по лестнице 100 427 (9,1%)
* 1 = Бег трусцой 342 177 (31,2%)
* 2 = Сидение 59 939 (5,5%)
* 3 = Стояние 48 395 (4,4%)
* 4 = Подъем по лестнице 122 869 (11,2%)
* 5 = Ходьба 424 400 (38,6%)
